# Elirox Agent Kit

Skill for using **Elirox** from any AI agent (Claude Code, Codex, Cursor, OpenCode, …).

## Install the skill

Paste this **single message** to your agent:

```
npx skills add k-elirox-agent/mcp, connect Elirox
```

The agent will install the skill and immediately ask for your API key — no extra steps.
